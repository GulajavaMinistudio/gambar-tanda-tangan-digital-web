# Screenshots — Panduan Penamaan File

Simpan screenshot aplikasi TTDigital di folder ini dengan nama file berikut.

## Daftar File yang Diperlukan

| Nama File                  | Konten Screenshot                | Format |
| -------------------------- | -------------------------------- | ------ |
| `01-main-menu.png`         | Halaman menu utama               | PNG    |
| `02-workspace-empty.png`   | Kanvas kosong (belum digambar)   | PNG    |
| `03-workspace-drawing.png` | Kanvas berisi tanda tangan       | PNG    |
| `04-color-picker.png`      | Bottom sheet pilih warna         | PNG    |
| `05-save-dialog.png`       | Bottom sheet simpan tanda tangan | PNG    |
| `06-help.png`              | Bottom sheet Bantuan             | PNG    |
| `07-about.png`             | Bottom sheet Tentang Aplikasi    | PNG    |

## Spesifikasi Screenshot

- **Resolusi**: Minimal 750×1624 px (portrait)
- **Perangkat**: Android (status bar hijau TTDigital)
- **Format**: PNG

## Cara Export dari Emulator / Device

1. Jalankan aplikasi di emulator atau device fisik
2. Navigate ke tampilan yang ingin di-screenshot
3. Tekan tombol screenshot (atau `Ctrl+S` di Android Studio emulator)
4. Rename file sesuai tabel di atas
5. Salin ke folder ini: `web_landingpage/assets/screenshots/`

## Catatan

Setelah menyimpan semua file, buka `web_landingpage/index.html` di browser
untuk memastikan carousel berjalan dengan baik.
